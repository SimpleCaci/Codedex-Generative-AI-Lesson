# Codedex Generative AI Lessons

A sequence of Python exercises completed while following the Codédex Generative AI learning pathway.

## Tech stack

- Python
- NumPy
- scikit-learn
- TextBlob
- Introductory natural-language processing concepts

## Lesson index

1. Generative AI introduction
2. Language-model fundamentals
3. Tokenization
4. N-grams
5. Text classification
6. Machine translation
7. Spell checking
8. A small movie-review sentiment experiment

## Run locally

```bash
python -m venv .venv
# activate the environment
pip install -r requirements.txt
python 08-WhatAreTheVibes.py
```

Individual lesson files are intentionally small and can be run independently.

## What I learned

The course introduced how text becomes model input, how simple statistical language techniques work, and how classification tools can turn text into practical predictions. The exercises are educational demonstrations rather than production AI systems.

## Repository maintenance

The generated virtual environment was removed from the current tree and replaced with a dependency manifest so the lessons are easier to review and reproduce.
